# 🌌 Acerca del proyecto

**Vehicle Rental Management System** es una aplicación de escritorio enfocada en la administración integral de procesos de renta de vehículos.

La plataforma permite:

- 🚗 Registrar vehículos
- 👥 Gestionar clientes
- 📅 Verificar disponibilidad
- 💰 Calcular pagos y cargos
- 🧾 Generar recibos
- 📊 Administrar rentas
- ⚡ Automatizar procesos
- 🏢 Gestionar operaciones empresariales

El sistema fue desarrollado utilizando arquitectura en capas para mantener una estructura modular y escalable.
